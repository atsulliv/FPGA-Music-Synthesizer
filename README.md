# FPGA-Music-Synthesizer

Final project code, worked alongside Jimi Long, Joseph Shull

I implemented 9 of the 18 total points added with our extensions to the FPGA music player and waveform display, which includes harmonics, dynamics, stereo, and PWM LEDs.

FPGA Used: Zynq-7000 SoC XC7Z020-1CLG400C
